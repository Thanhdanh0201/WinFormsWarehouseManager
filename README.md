#  WarehouseManager 

##  Giới thiệu

Đây là hệ thống quản lý kho hàng (Warehouse Management System - WMS) được thiết kế để tối ưu hóa các hoạt động kho chứa như nhập, xuất, kiểm kê và theo dõi hàng tồn kho.

##  Công Nghệ Sử Dụng

* **Backend:** C# 
* **Frontend:** WinForms 
* **Database:** MySQL
