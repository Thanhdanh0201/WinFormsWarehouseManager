#  WarehouseManager 

##  Giới thiệu

Đây là hệ thống quản lý kho hàng (Warehouse Management System - WMS) được thiết kế để tối ưu hóa các hoạt động kho chứa như nhập, xuất, kiểm kê và theo dõi hàng tồn kho.

##  Công Nghệ Sử Dụng

* **Backend:** C# 
* **Frontend:** WinForms 
* **Database:** MySQL 

## Cài Đặt và Khởi Chạy

1.  Clone repository này về máy của bạn: `git clone [URL_Dự_án_của_bạn]`
2.  Mở project trong Visual Studio.
3.  Cập nhật chuỗi kết nối database trong file `App.config` (hoặc `appsettings.json`).
4.  Build và Run.
